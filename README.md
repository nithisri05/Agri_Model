# AI-Powered Smart Agriculture Suite

## Overview

This project is a comprehensive **AI-driven agriculture decision support system** designed to address real-world farming challenges using Machine Learning and Explainable AI.

It consists of **three interconnected notebooks**, each solving a critical problem in modern agriculture:

1. Crop Yield Prediction & Advisory System
2. Sustainable Resource Optimization System
3. AI-Powered Crop Recommendation Agent

Together, these systems provide **data-driven insights, predictions, and actionable recommendations** to improve agricultural productivity and sustainability.

---

## Problem Statement

Farmers often face challenges such as:

* Uncertain crop yield
* Inefficient use of water and fertilizers
* Difficulty in selecting suitable crops
* Lack of data-driven decision support

This project aims to bridge that gap by building **intelligent AI systems** that assist in decision-making.

---

## Project Architecture

```
Input Data → Data Processing → Feature Engineering → ML Models → Explainable AI → Decision System → User Interface
```

---

# Notebook-1: Crop Yield Prediction & Advisory

## Objective

Predict crop yield based on farm-level inputs and provide recommendations to improve productivity.

## Key Features

* Regression models (Linear, Decision Tree, Random Forest)
* Feature engineering (efficiency-based features)
* Model evaluation (MAE, RMSE, R²)
* Hyperparameter tuning
* Explainable AI (SHAP)
* Recommendation system
* Gradio-based interactive UI

## Key Insights

* Resource efficiency (water & fertilizer per acre) is more impactful than raw usage
* Yield is influenced by multiple external factors beyond dataset scope

---

# Notebook-2: Sustainable Agriculture Optimization

## Objective

Analyze and optimize resource usage to promote sustainable farming practices.

## Key Features

* Focus on water and fertilizer optimization
* Detection of inefficient farming practices
* Custom sustainability metrics
* Machine learning models for pattern detection
* Explainable AI for understanding inefficiencies
* Actionable recommendations for resource management

## Key Insights

* Excess resource usage does not guarantee higher yield
* Balanced input utilization leads to better sustainability

---

# Notebook-3: Crop Recommendation AI Agent

## Objective

Recommend the most suitable crop based on soil nutrients and provide intelligent advisory.

## Key Features

* Classification models (Logistic, Decision Tree, Random Forest)
* Soil-based feature engineering (NPK, fertility score)
* Explainable AI for crop selection reasoning
* AI agent simulation for interactive recommendations
* Optional Gradio interface

## Key Insights

* Soil nutrient balance directly impacts crop suitability
* Intelligent recommendations improve farming decisions
